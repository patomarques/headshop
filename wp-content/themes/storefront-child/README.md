# Storefront Child Theme

Este é um tema filho para o tema Storefront do WooCommerce.

## Como usar
1. Copie esta pasta para `wp-content/themes/`.
2. Ative o tema filho no painel do WordPress.
3. Adicione suas customizações em `style.css` e `functions.php`.

## Estrutura
- `style.css`: Informações do tema e customizações de CSS.
- `functions.php`: Funções e scripts do tema filho.
- `screenshot.png`: Imagem de preview do tema (adicione sua própria imagem).
